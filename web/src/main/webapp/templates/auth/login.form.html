Interoperability through Linked Open Data (LOD) Cloud

Within the Linked Open Data cloud, each MDR maintains a set of Common Data Elements (CDE) together with the corresponding components and relations. The CDEs are linked to CDEs of other MDRs.

Federated Semantic MDR Framework

Linked Open Data integration of data element repositories where each element can be a uniquely referenced and processed to enable the syntactic and semantic interoperability.

SALUS Project

Extraction Specification

Semantic MDR can provide extraction specifications for a data element defined in a selected domain (like CDISC SDTM data elements), from an implementation dependant content model in another domain (like HL7/ASTM CCD).

Development
