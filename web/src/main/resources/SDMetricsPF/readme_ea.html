Description

This archive provides an XMI transformation file for the SDMetrics default metamodel for UML1.x and XMI 1.2 files saved by Enterprise Architect.

SDMetrics' default XMI transformations expect all model elements in the XMI file to have and XMI ID. Enterprise Architect does not provide XMI IDs for all model elements - this specialized transformation file accounts for this.
